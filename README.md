**Project Overview**
This project analyzes retail sales data to uncover key business insights such as sales performance, profit trends, customer behavior, and regional performance.
The dashboard is built using Power BI and helps stakeholders make data-driven decisions.

**Objectives**
Analyze total sales and profit
Identify top-performing categories and products
Understand regional performance
Track customer segments and trends
Detect loss-making areas


**Dataset Details**
Dataset Name: Superstore Sales Dataset
Total Records: ~300 rows (sample dataset)
Columns Include:
Order Date
Region
Category & Sub-category
Sales
Profit
Customer Segment


**Tools & Technologies Used**
Power BI (Data Visualization)
Excel (Data Cleaning - if used)
DAX (for calculations)


**Key Dashboards Created**
Sales Overview Dashboard
Profit Analysis
Regional Performance
Category-wise Analysis
Customer Segment Insights
